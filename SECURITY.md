# Security

## Reporting a Vulnerability

If you find a security issue, please don't open a public GitHub issue.

Email: hi@adsbar.app

I'll respond within 48 hours and aim to fix confirmed issues within 7 days.

## Scope

ADSBar communicates only with devices on your local network. No data is sent to external servers.

## Out of Scope

- Issues requiring physical access to your network
- Vulnerabilities in third-party ADS-B firmware (readsb, dump1090, tar1090)
